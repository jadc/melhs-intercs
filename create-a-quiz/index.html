Glorious Javascript Quiz

Question #1

What does the following output?

4 - "2"

The answer was: 2

An integer subtracting a string with an integer will treat the string like an integer and perform the arithmetic.

Question #2

What does the following output?

9 + "10"

The answer was: "910"

An integer added to a string will append the string to the integer.

Question #3

What does the following output?

0.2 + 0.2 == 0.4

The answer was: true

This is just how math works...
